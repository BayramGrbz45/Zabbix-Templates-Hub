# Zabbix Templates Hub
 
Özel Zabbix izleme template'lerinden oluşan bir koleksiyon.
 
## Kurulum
 
1. İhtiyacınız olan template dosyasını (`.yaml` / `.xml`) indirin.
2. Zabbix arayüzünde: **Data collection → Templates → Import**.
3. Dosyayı yükleyin.
4. Template'i host'a bağlayın ve gerekli macro'ları ayarlayın.
